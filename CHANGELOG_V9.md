# Raccoon Signal v9

## Public experience

- Refined the working brand to **Raccoon Signal**, while keeping the registered legal entity configurable separately.
- Rebuilt the public English/Finnish landing page around plain-language outcomes instead of technical terminology.
- Added consistent bilingual navigation, language controls, service packages, process explanations, estimate flow, contact forms and client support.
- Moved existing-client requests to a dedicated no-login support page using private project codes.
- Redesigned company, privacy, cookie, service terms, accessibility and data-processing pages with clearer navigation and readable layouts.

## Private workspace

- Reorganised the administrator dashboard and Business suite into clearer task groups.
- Standardised headings, buttons, spacing, cards, status elements and responsive behaviour across private modules.
- Improved the terminal map's visibility, marker clustering, keyboard navigation and mouse-wheel handling while preserving the black-and-green style.
- Updated login wording so it is clearly a staff-only area and never appears to be a customer portal.

## Reliability and accessibility

- Added a skip link, visible keyboard focus, reduced-motion support and improved responsive navigation.
- Kept all core tools independent of Claude; optional AI assistance continues to use a deterministic fallback.
- Preserved server-side validation, role restrictions and production security headers.
